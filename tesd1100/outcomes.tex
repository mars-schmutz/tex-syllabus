Students will be prepared to obtain entry-level employment in the software engineering field and will demonstrate:
\begin{itemize}
    \item Explore modern software development.
    \item Deliver software applications using current project management practices.
    \item Demonstrate problem-solving skills with computer programming.
    \item Practice workplace professionalism.
    \item Demonstrate the use of techniques and tools necessary for Software Development.
\end{itemize}
